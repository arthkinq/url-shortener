# URL Shortener API Documentation

This document describes the available endpoints, expected request payloads, and response formats for the URL Shortener REST API. The API is designed to be easily integrated into web applications, browser extensions, or third-party services.

## Base URL
All API requests (except for the redirect endpoint) should be prefixed with the API version path:
`http://<your-domain-or-ip>/api/v1`

---

## 1. Create a Short URL

Creates a new short URL for the provided original URL.

* **URL:** `/api/v1/urls`
* **Method:** `POST`
* **Content-Type:** `application/json`

### Request Body
| Field | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `original_url` | `string` | The full HTTP/HTTPS URL you want to shorten. | Yes |

**Example Request:**
```json
{
  "original_url": "https://github.com/arthkinq/ww2-interactive-map"
}
```

### Success Response
* **Code:** `201 Created`
* **Content:**
```json
{
  "short_id": "xG7p9A",
  "short_url": "http://localhost:8000/xG7p9A"
}
```

### Error Responses
* **Code:** `422 Unprocessable Entity`
  * **Reason:** The provided URL is invalid (e.g., missing `http://` or `https://`).
* **Code:** `500 Internal Server Error`
  * **Reason:** Failed to generate a unique ID due to a severe database collision.

---

## 2. Redirect to Original URL

Redirects the client to the original URL and increments the internal click counter. This endpoint is mapped to the root of the domain.

* **URL:** `/{short_id}`
* **Method:** `GET`

### URL Parameters
| Parameter | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `short_id` | `string` | The unique 6-character Base62 string generated by the API. | Yes |

### Success Response
* **Code:** `307 Temporary Redirect`
* **Headers:** `Location: <original_url>`
  * *Note: 307 is used instead of 301 to ensure the browser does not permanently cache the redirect, allowing accurate click tracking.*

### Error Responses
* **Code:** `404 Not Found`
  * **Reason:** The requested `short_id` does not exist in the database.

---

## 3. Get URL Statistics

Retrieves the current statistics for a specific shortened URL, including total clicks and creation date.

* **URL:** `/api/v1/urls/{short_id}/stats`
* **Method:** `GET`

### URL Parameters
| Parameter | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `short_id` | `string` | The unique 6-character Base62 string of the short URL. | Yes |

### Success Response
* **Code:** `200 OK`
* **Content:**
```json
{
  "original_url": "https://github.com/arthkinq/ww2-interactive-map",
  "short_id": "xG7p9A",
  "clicks": 42,
  "created_at": "2026-07-02T13:25:00.123456Z"
}
```

### Error Responses
* **Code:** `404 Not Found`
  * **Reason:** The requested `short_id` does not exist in the database.
